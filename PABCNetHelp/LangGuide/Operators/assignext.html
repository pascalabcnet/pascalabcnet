Расширенные операторы присваивания

Расширенный оператор присваивания имеет вид:

переменная op выражение

где op - один из значков расширенного присваивания +=, -=, *=, /=.

В качестве переменной может быть простая переменная, разыменованный указатель, переменная с индексами или компонент переменной типа запись, а также свойство класса.

Расширенные операторы присваивания имеют следующий смысл: a #= b означает a := a # b, где # - знак операции +, -, *, /.

Расширенные операторы присваивания используются прежде всего для числовых типов. Например:

a += 3; // a := a + 3; - увеличить a на 3
b *= 2; // b := b * 2; - увеличить b в 2 раза

Оператор /= неприменим если левый операнд - целый.
