Методы расширения словарей

function Each<Key,Source,Res>(Self: sequence of System.Linq.IGrouping<Key,Source>; grOperation: System.Linq.IGrouping<Key,Source> -> Res): Dictionary<Key,Res>;
        Возвращает словарь, сопоставляющий ключу группы результат групповой операции
function Each<Key,Res>(Self: sequence of Key; proj: Key -> Res): Dictionary<Key,Res>;
        Возвращает словарь, сопоставляющий элементам последовательности определённые значения
function EachCount<Key,Source>(Self: sequence of System.Linq.IGrouping<Key,Source>): Dictionary<Key,integer>;
        Возвращает словарь, сопоставляющий ключу группы количество элементов с данным ключом
function EachCount<T>(Self: sequence of T): Dictionary<T,integer>;
        Возвращает частотный словарь объектов последовательности
function Get<Key, Value>(Self: IDictionary<Key, Value>; K: Key; V: Value := default(Value)): Value;
        Возвращает в словаре значение, связанное с указанным ключом, а если такого ключа нет, то значение по умолчанию
function GetOrAdd<Key, Value>(Self: IDictionary<Key, Value>; K: Key; valueFactory: Key -> Value): Value;
        Получает значение по ключу или добавляет новое, если ключ отсутствует
function GetOrAdd<Key, Value>(Self: IDictionary<Key, Value>; K: Key; defaultValue: Value): Value;
        Получает значение по ключу или добавляет значение по умолчанию
function operator-<TKey, TVal>(Self: Dictionary<TKey, TVal>; key: TKey): Dictionary<TKey, TVal>;
        Возвращает словарь, в котором из исходного словаря удален элемент с данным ключом
function operator-<TKey, TVal>(Self: Dictionary<TKey, TVal>; keys: sequence of TKey): Dictionary<TKey, TVal>;
        Возвращает словарь, в котором из исходного словаря удалены все элементы с ключами, задаваемыми вторым операндом
function operator+<TKey, TVal>(Self: Dictionary<TKey, TVal>; dict: Dictionary<TKey, TVal>): Dictionary<TKey, TVal>;
        Объединяет данные в двух словарях. Если в обоих имеются одинаковые ключи, то с ключом связывается значение из второго словаря
procedure operator+=<TKey, TVal>(Self: Dictionary<TKey, TVal>; update: Dictionary<TKey, TVal>);
        Обновляет данные в словаре данными из другого словаря
procedure operator-=<Key,Value>(Self: IDictionary<Key,Value>; k: Key);
        Удаляет из словаря пары с указанным значением ключа
procedure operator-=<Key,Value>(Self: IDictionary<Key,Value>; keys: sequence of Key);
        Удаляет из словаря пары с указанными значениями ключа
procedure Update<TKey, TVal>(Self: Dictionary<TKey, TVal>; update: Dictionary<TKey, TVal>);
        Обновляет данные в словаре данными из другого словаря
